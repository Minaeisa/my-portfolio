# Public Assets

ضع هنا صورة البروفايل التي ستظهر في الهيدر.

• اسم الملف المطلوب: `profile.jpg`
• يمكن استخدام `jpg` أو `png`، ويفضل أبعاد مربعة (مثلاً 512x512)
• الخلفية الشفافة اختيارية؛ سيتم قص الصورة تلقائيًا داخل دائرة

بعد وضع الملف، افتح `http://localhost:3000` للتأكد من ظهوره في قسم الـ Hero.
